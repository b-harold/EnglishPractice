# English Pronunciation Trainer

Small single-page web app to practice English pronunciation using the browser Speech APIs.

## Features
- Listen to example phrases using Text-to-Speech (TTS).
- Record your pronunciation and get a basic similarity score (Levenshtein-based).
- Add your own practice phrases.

## Requirements
- Modern desktop browser with Web Speech API support (Google Chrome, Microsoft Edge).

## Usage
1. Open `index.html` in a supported browser (double-click or serve via a static server).
2. Click `Listen` to hear the target phrase.
3. Click `Practice` and speak; the app will transcribe and show an accuracy score.

Note: Speech recognition requires microphone access and may vary by browser/device.
